 Cabzo – Car Booking App

Cabzo is a user-friendly mobile application developed using Android Studio that allows users to easily book car rides through a simple and intuitive interface. The app is designed to simulate a real-world ride booking system where users can select cars, view ride details, and manage their bookings efficiently.

 Features

Easy car booking interface
View available cars and ride details
Simple and user-friendly mobile UI
Smooth navigation between app screens
Lightweight and responsive Android application

 Technologies Used

Java 
Android Studio
XML (for UI design)

 How to Run

Clone the repository
Open the project in Android Studio
Build and run the app using an Android emulator or physical device

Purpose
This project was developed to practice Android mobile application development and to understand core concepts such as UI design, activity navigation, and basic booking system functionality.
